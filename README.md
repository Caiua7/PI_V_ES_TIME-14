# NeoPrice
Projeto com foco em frontend para replicar/adaptar funcionalidades do portal Neo Price.

## Entrega atual
- Frontend implementado com 6 telas.
- Backend nao implementado (somente especificacoes tecnicas e contratos).

## Como rodar
```bash
cd frontend
npm install
npm run dev
```
